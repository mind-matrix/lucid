# @mind-matrix/lucid-components

Framework-agnostic UI components built on Web Components. Accessibility-first (WCAG 2.1 AA) and internationalization-ready by design. Works in **React, Vue, Angular, Astro, Svelte, and vanilla HTML** — no wrappers, no runtime, no lock-in.

Under 4 KB gzipped for every shipped component combined.

## Install

```sh
bun add @mind-matrix/lucid-components
```

## Use

Import once at your app's entry, then use the custom elements anywhere in your markup:

```ts
import "@mind-matrix/lucid-components";
```

## Optional design tokens

```html
<link rel="stylesheet" href="@mind-matrix/lucid-core/tokens.css" />
```
