# @mind-matrix/lucid-core

Runtime primitives that power [Lucid components](https://www.npmjs.com/package/@mind-matrix/lucid-components). Most consumers get this transitively via `@mind-matrix/lucid-components` — you only install it directly if you're **authoring your own Lucid-style components** or want the design tokens without the components.

Under 2 KB gzipped.

## Install

```sh
bun add @mind-matrix/lucid-core
```

## Design tokens

```html
<link rel="stylesheet" href="@mind-matrix/lucid-core/tokens.css" />
```

## Documentation

Primitives reference, JSX runtime setup, tokens list, and authoring guide: **<https://mind-matrix.github.io/lucid>**
